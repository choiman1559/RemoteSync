## Remote Actions

Syncprotocol can perform various "remote actions" like:

 - Check other device battery status
![battery.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/battery.gif?raw=true)

 - Move files from pc to mobile
![file_pc_to_mobile.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/file_pc_to_mobile.gif?raw=true)

 - Or vice versa:
![file_mobile_to_pc.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/file_mobile_to_pc.gif?raw=true)

 - Share image from web directly
![img_mobile_to_pc.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/img_mobile_to_pc.gif?raw=true)

 - Copy text to clipboard
![clip_mobile_to_pc.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/clip_mobile_to_pc.gif?raw=true)

 - Use mobile device as remote control for presentation
![ppt_mobile_to_pc.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/ppt_mobile_to_pc.gif?raw=true)

 - Open web link at another device
![web_mobile_to_pc.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/web_mobile_to_pc.gif?raw=true)
![web_pc_to_mobile.gif](https://github.com/choiman1559/RemoteSync/blob/master/docs/web_pc_to_mobile.gif?raw=true)

In addition to the above, there are many tasks such as:
-   Send notification with custom title and message
-   Trigger tasker event
-   Run application with package name
-   Run terminal command
-   Media remote control (like mpris)
